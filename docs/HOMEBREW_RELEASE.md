# Homebrew release guide

This guide describes how to release `speq-cli` and publish/update Homebrew install.

## 1) Create GitHub release assets

Release assets are generated by `.github/workflows/release.yml`.

### Option A: tag push

```bash
git tag v0.1.0-alpha.0
git push origin v0.1.0-alpha.0
```

### Option B: manual dispatch

Run workflow `release` with:

- `tag`: `v0.1.0-alpha.0`
- `prerelease`: `true` (or `false` for stable)

Expected artifacts in the release:

- `speq-darwin-aarch64.tar.gz`
- `speq-darwin-x86_64.tar.gz`
- `speq-linux-x86_64.tar.gz`
- `*.sha256` for each tarball

## 2) Prepare Homebrew formula content

Collect SHA256 values from release artifacts:

```bash
curl -fsSL https://github.com/speq-tms/speq-cli/releases/download/v0.1.0-alpha.0/speq-darwin-aarch64.tar.gz.sha256
curl -fsSL https://github.com/speq-tms/speq-cli/releases/download/v0.1.0-alpha.0/speq-darwin-x86_64.tar.gz.sha256
```

Generate formula text:

```bash
DARWIN_ARM_SHA256="<arm_sha256>" \
DARWIN_AMD_SHA256="<x86_sha256>" \
./scripts/generate_homebrew_formula.sh 0.1.0-alpha.0 > /tmp/speq.rb
```

## 3) Update Homebrew tap

In `speq-tms/homebrew-tap`:

1. Replace `Formula/speq.rb` with generated content.
2. Commit and push.

Example:

```bash
cd /path/to/homebrew-tap
cp /tmp/speq.rb Formula/speq.rb
git add Formula/speq.rb
git commit -m "speq 0.1.0-alpha.0"
git push origin main
```

## 4) Validate installation

```bash
brew untap speq-tms/tap || true
brew tap speq-tms/tap https://github.com/speq-tms/homebrew-tap
brew update
brew reinstall speq
speq --help
```
