# Homebrew release guide

This guide describes how to release `speq-cli` and publish/update Homebrew install.

## Release branch workflow

For each upcoming version, use a dedicated candidate branch and keep `main` release-only.

Example for `v0.1.0-alpha.1`:

1. Create candidate branch from `main`: `v0.1.0-alpha.1`.
2. Create delivery branches from candidate branch:
   - `feat/<scope>`
   - `fix/<scope>`
   - `chore/<scope>`
3. Open PRs into `v0.1.0-alpha.1` until release scope is complete.
4. Open final PR from `v0.1.0-alpha.1` to `main`.
5. Tag `main` with `v0.1.0-alpha.1` and run `release` workflow.

Required checks on PRs (recommended):

- `ci`
- `mvp-integration`
- `pr-title` (Conventional Commit style PR title)

## 1) Create GitHub release assets

Release assets are generated by `.github/workflows/release.yml`.

### Option A: tag push

```bash
git tag v0.1.0-alpha.0
git push origin v0.1.0-alpha.0
```

### Option B: manual dispatch

Run workflow `release` with:

- `tag`: `v0.1.0-alpha.0`
- `prerelease`: `true` (or `false` for stable)

Expected artifacts in the release:

- `speq-darwin-aarch64.tar.gz`
- `speq-darwin-x86_64.tar.gz`
- `speq-linux-x86_64.tar.gz`
- `*.sha256` for each tarball

## Automation prerequisites

To enable automatic Homebrew tap PR creation after `speq-cli` release:

1. Add repository secret in `speq-cli`:
   - `HOMEBREW_TAP_PAT` (PAT with `repo` access to `speq-tms/homebrew-tap`).
2. Ensure `speq-tms/homebrew-tap` has workflow `update-formula-from-release`.
3. Keep `homebrew-tap` branch protection enabled for PR-only changes on `main`.

## 2) Prepare Homebrew formula content

Collect SHA256 values from release artifacts:

```bash
curl -fsSL https://github.com/speq-tms/speq-cli/releases/download/v0.1.0-alpha.0/speq-darwin-aarch64.tar.gz.sha256
curl -fsSL https://github.com/speq-tms/speq-cli/releases/download/v0.1.0-alpha.0/speq-darwin-x86_64.tar.gz.sha256
```

Generate formula text:

```bash
DARWIN_ARM_SHA256="<arm_sha256>" \
DARWIN_AMD_SHA256="<x86_sha256>" \
./scripts/generate_homebrew_formula.sh 0.1.0-alpha.0 > /tmp/speq.rb
```

## 3) Update Homebrew tap

Preferred: automatic update from `speq-cli` release workflow.

- `speq-cli` release emits `repository_dispatch` (`speq-cli-release`) to `speq-tms/homebrew-tap`.
- `homebrew-tap` workflow `update-formula-from-release` updates `Formula/speq.rb` and opens a PR.
- Merge that PR after `homebrew-tap` CI is green.

Fallback (manual) remains available:

1. Replace `Formula/speq.rb` with generated content.
2. Commit and push.

## 4) Validate installation

```bash
brew untap speq-tms/tap || true
brew tap speq-tms/tap https://github.com/speq-tms/homebrew-tap
brew update
brew reinstall speq
speq --help
```
